# 🏚️ 废土求生 (Wasteland Survive)

> 核战后的第三十天。天空永远是灰蒙蒙的，空气中弥漫着尘埃和腐朽的气息。
> 你从避难所的废墟中醒来，外面的世界已经面目全非。
> 你的食物和水所剩无几。在废墟中，也许还能找到生存的物资。
> 你听说城市的东边有一座通讯塔，如果能修好它，也许能联系到其他幸存者……
> 但首先，你需要活下去。

---

## 🎮 游戏简介

《废土求生》是一款末日生存文字游戏，融合了像素帧动画与CRT复古风格。在这片核战后的废墟中，你将面对饥饿、口渴、疾病、辐射和敌对幸存者的威胁。你需要探索废墟、收集物资、合成装备、与其他幸存者交易或对抗，最终找到无线电零件向外界求救。

**游戏类型：** 文字冒险 + 生存模拟 + 像素艺术
**游戏时长：** 约30-60分钟（一局）
**难度：** 中等偏高（肉鸽元素，随机事件）

---

## 🌍 世界观

2024年，全球核战争爆发。在短短几小时内，城市化为废墟，文明崩塌。三十天后，你从地下避难所的废墟中苏醒。外面的世界已经面目全非——天空永远是灰蒙蒙的，空气中弥漫着辐射尘埃，地面上到处是变异生物和敌对幸存者。

但并非所有人都死了。在这片废土上，还有其他幸存者在挣扎求生。有些人选择合作，有些人选择掠夺。你需要在这些复杂的人际关系中找到生存之道。

## 🎯 核心玩法

### 生存循环
- **每回合 = 1天**
- 每天你需要做出选择：探索、休息、合成、使用物品、移动
- 管理你的生存指标，避免任何一项归零
- 应对随机事件：遭遇战、天气灾害、陷阱、线索

### 目标
收集 **3个无线电零件** → 前往 **通讯塔** → 修复通讯设备 → 等待救援

## 💚 生存系统

| 指标 | 初始值 | 每天消耗 | 归零后果 | 恢复方式 |
|------|--------|----------|----------|----------|
| ❤️ 生命 (HP) | 100 | - | **死亡** | 药品、绷带、休息 |
| 🍖 饥饿 | 100 | -15 | HP-10/天 | 食物 |
| 💧 口渴 | 100 | -20 | HP-15/天 | 水 |
| ⚡ 精力 | 100 | -10 | 无法探索 | 休息 |
| 🧠 理智 | 100 | -5 | 随机负面事件 | 休息、祝福 |

**生存技巧：**
- 饥饿和口渴消耗最快，优先保证食物和水供应
- 精力不足时无法探索，合理安排休息时间
- 理智过低会触发精神崩溃事件，损失物资或生命

### NPC交互方式

- **💬 对话：** 了解NPC的故事和背景
- **💰 交易：** 用食物换取物资，或出售多余物品
- **⚔️ 雇佣：** 雇佣安娜作为佣兵，提升战斗力
- **💊 治疗：** 找老陈治疗伤势，恢复HP
- **🙏 祝福：** 找玛丽亚恢复理智
- **🔧 升级：** 找刘瘸子升级武器或护甲
- **🎯 清剿：** 雇佣赵四清除地点的敌对势力
- **🌿 草药园：** 激活林小雨的草药园，每天自动获得草药
- **🔍 情报：** 从张大力获取教会内部情报
- **⛏️ 拾荒：** 跟老王去辐射区寻找稀有物资

## ⚔️ 战斗系统

### 遭遇敌人
探索时有几率遭遇敌对幸存者或变异生物。遭遇率受以下因素影响：
- 地点风险等级
- 是否携带火把（降低50%遭遇率）
- 天数（随天数增加而增加）

### 战斗选项
| 选项 | 效果 |
|------|------|
| ⚔️ 攻击 | 对敌人造成伤害，但也会受到反击 |
| 🛡️ 防御 | 减少受到的伤害，消耗少量精力 |
| 🏃 逃跑 | 60%几率成功逃脱，失败则受到重创 |

### 战斗奖励
击败敌人后可以获得物资奖励，包括食物、水、金属碎片等。

## 🏆 通关条件

1. **收集3个无线电零件**
   - 通过探索获得
   - 通过"发现线索"事件获得
   - 从阿光或老王处购买

2. **前往通讯塔**
   - 通讯塔是最高风险地点
   - 建议装备武器和护甲
   - 带足食物和药品

3. **完成通讯塔保卫战**
   - 启动通讯设备会引来掠夺者
   - 需要击退3波进攻
   - 最终BOSS是掠夺者首领

4. **等待救援**
   - 成功击退所有敌人后，通讯设备启动
   - 北方幸存者基地回应
   - 游戏通关！

## 💡 生存技巧

1. **前期优先探索超市和居民区**，风险较低，物资丰富
2. **尽快合成武器和护甲**，提升战斗力
3. **携带火把探索**，降低遭遇率
4. **与NPC建立良好关系**，获取特殊服务
5. **合理安排休息时间**，避免精力耗尽
6. **储备足够的食物和水**，这是最紧缺的资源
7. **升级避难所设施**，获得稳定的资源产出
8. **在挑战通讯塔前做好充分准备**，带足药品和食物

## 🔧 技术信息

- **后端：** Python FastAPI + SQLite (存档)
- **前端：** HTML + CSS + Canvas像素动画
- **部署：** Docker Compose / 本地 uvicorn

### 运行方式

```bash
# 本地运行
cd ~/wasteland-survive
.venv/bin/uvicorn backend.main:app --host 0.0.0.0 --port 8000

# Docker运行
cd ~/wasteland-survive
docker compose up --build
```

浏览器打开 `http://localhost:8000` 即可游玩。
