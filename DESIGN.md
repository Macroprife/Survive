# 废土求生 (Wasteland Survive)

末日生存文字游戏 + 像素帧动画

## 核心玩法

- 每回合 = 1天
- 选择行动：探索、休息、合成、使用物品、移动
- 管理生存指标：生命/饥饿/口渴/精力/理智
- 随机事件系统（遭遇战、天气、陷阱、线索）
- 目标：收集3个无线电零件 → 前往通讯塔 → 求救通关

## 生存指标

| 指标 | 初始值 | 每天消耗 | 归零后果 |
|------|--------|----------|----------|
| HP   | 100    | -        | 死亡     |
| 饥饿 | 100    | -15      | HP-10/天 |
| 口渴 | 100    | -20      | HP-15/天 |
| 精力 | 100    | -10      | 无法探索 |
| 理智 | 100    | -5       | 随机负面事件 |

## 地点

| 地点      | 风险 | 可搜物资        |
|-----------|------|-----------------|
| 庇护所    | 低   | 布料、木头      |
| 超市      | 中   | 食物、饮料、酒精|
| 医院      | 高   | 药品、绷带、草药|
| 警察局    | 高   | 武器、护甲、金属|
| 住宅区    | 中   | 杂物、材料、零件|
| 无线电塔  | 极高 | 无线电零件      |

## 合成配方

| 配方       | 材料              | 效果     |
|------------|-------------------|----------|
| 草药药剂   | 草药x1 + 水x1    | HP +30   |
| 消毒绷带   | 布料x1 + 酒精x1  | HP +20   |
| 自制武器   | 金属x2 + 木头x1  | 攻击 +8  |
| 强化护甲   | 金属x3 + 布料x2  | 防御 12  |
| 火把       | 木头x1 + 布料x1  | 降低遭遇率|

## 按键操作

| 按键 | 功能 |
|------|------|
| E / 1 | 探索当前地点 |
| R / 2 | 休息 |
| C / 3 | 合成菜单 |
| I / 4 | 使用物品 |
| M / 5 | 地点导航 |
| Ctrl+S | 保存游戏 |
| H / ? | 帮助 |
| A | 攻击（战斗中）|
| D | 防御（战斗中）|
| F | 逃跑（战斗中）|
| Esc | 关闭弹窗 |

## 像素动画

- 角色idle呼吸动画
- 天气效果(雨/沙尘暴)
- 篝火/营地
- 探索场景切换
- 战斗闪帧
- CRT扫描线效果

## 技术栈

- **后端**: Python FastAPI + SQLite (存档)
- **前端**: HTML + CSS + Canvas像素动画
- **部署**: Docker Compose / 本地 uvicorn

## 运行方式

```bash
# 本地运行
cd ~/wasteland-survive
.venv/bin/uvicorn backend.main:app --host 0.0.0.0 --port 8000

# Docker 运行 (需要 Docker 环境)
cd ~/wasteland-survive
docker compose up --build
```

浏览器打开 `http://localhost:8000` 即可游玩。
